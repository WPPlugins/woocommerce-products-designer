<h4>Overview</h4>
The Woocommerce Products Designer (WPD) is a woocommerce extension which allows your customers to create their perfect logo, t-shirts, business cards, prints, mugs and any product which needs to be created online prior to order.

The Woocommerce Products Designer is built using WordPress best practices both on the front and the back end. This results in an efficient, robust and intuitive plugin. It's works with any theme, including the default WordPress themes and the last Woocommerce version.

<h4>Features</h4>
<ul>
	<li>HTML5 canvas technology</li>
	<li>Unlimited customizable parts/pages</li>
	<li>Integrated google fonts loader</li>
	<li>Categorizable cliparts library</li>
	<li>Drag and drop editor</li>
	<li>PDF output</li>
	<li>Professional support</li>
</ul>

<h4>Pro features</h4>
<ul>
	<li>Templates creation available</li>
	<li>Editor configurable dimensions</li>
	<li>Save designs for later</li>
	<li>Configurable minimum uploads dimensions</li>
	<li>SVG files edition</li>
	<li>Configurable colors palette</li>
	<li>Pictures extraction from social networks</li>
	<li>Ability to upload custom files for order</li>
	<li>Custom pricing rules</li>
	<li>Configurable output file resolution</li>
</ul>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of Woocommerce Product Designer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "Woocommerce Product Designer"ï¿½ and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>woocommerce-product-designer.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>woocommerce-product-designer.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>woocommerce-product-designer.zip</code></li>
<li>Extract the <code>woocommerce-product-designer</code> directory to your computer</li>
<li>Upload the <code>woocommerce-product-designer</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

<h4>Updating</h4>

<p>You can use automatic update to update the plugin safely.</p></div>
